Chargement des données MNIST

  • Le code utilise keras.datasets.mnist pour charger les ensembles d'entraînement et de test des images de chiffres manuscrits.
  • Les données sont divisées en train_images, train_labels (images et étiquettes d'entraînement) et test_images, test_labels (images et étiquettes de test).

Visualisation des données

  1. La distribution des classes est affichée à l'aide de pd.Series(train_labels).value_counts().sort_index().
  2. Un échantillon de 10 images est affiché avec leurs étiquettes correspondantes en utilisant matplotlib.pyplot.

Entraînement d'un modèle de classification

  • Un modèle séquentiel est créé avec des couches de convolution (Conv2D), de pooling (MaxPooling2D), d'aplatissement (Flatten), et de neurones fully connected (Dense).
  • Le modèle est compilé avec l'optimiseur Adam, la fonction de perte sparse_categorical_crossentropy, et la métrique accuracy.
  • Le modèle est entraîné sur les données d'entraînement avec validation sur les données de test.

Évaluation du modèle

Le modèle est évalué sur les données de test, affichant une précision de 99.22%.

Visualisation des prédictions

Les prédictions du modèle sont affichées pour les 10 premières images du jeu de test, avec les étiquettes réelles et prédites.
